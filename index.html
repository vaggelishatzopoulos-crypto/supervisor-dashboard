How it works
Three pillars · one composite score · every number is exact

Operational KPIs
Enter the actual number (e.g. LQA = 83). The score is calculated automatically using continuous linear interpolation — 83 and 87 are different scores. No bands, no ranges.
35%
Supervisor Competencies
10-point scale. Each point = exactly 10%. Score 7 = 70%, Score 8 = 80%. Full behavioral anchor shown for every level — click to see what each score means.
